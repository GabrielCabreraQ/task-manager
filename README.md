# Task Manager - Gestión de Tareas

Aplicación web para la gestión de tareas.

## Tecnologías utilizadas

- **Backend**: Golang + Gin + MongoDB
- **Frontend**: Nuxt3 + Vue3
- **Base de datos**: MongoDB
- **Despliegue**: docker + docker compose
- **CI/CD**: Jenkins

## Estructura del proyecto

task-manager/
├── task-backend/           # API REST en Go
├── task-frontend/          # Interfaz web en Nuxt 3 + Vue3
├── deployment/        # Archivos de despliegue
│   ├── docker-compose.yml
│   └── Jenkinsfile
├── README.md
